# Branch A: Personal Website

After the universal questions, ask these steps in order.

## Step 4｜First screen style

1. Full-screen image background
2. Large headline with a small image
3. Headline-only minimal layout
4. Split layout with text on one side and image on the other
5. Motion-led or video-led hero

## Step 5｜Navigation style

1. Very simple top navigation
2. Transparent floating navigation
3. Solid fixed navigation
4. Side navigation
5. Minimize navigation in the first screen

## Step 6｜Hero hierarchy

1. Name is largest
2. Role or identity is largest
3. Name and role are equally important
4. A value statement is largest

## Step 7｜Project presentation

1. Large visual showcases
2. Card grid
3. Vertical list
4. Timeline
5. Alternating editorial layout
