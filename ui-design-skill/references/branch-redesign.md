# Branch D: Redesign an Existing Page

After the universal questions, ask these steps in order.

## Step 4｜What must be preserved

1. Brand colors
2. Logo or brand recognition
3. Page structure
4. Existing copy
5. Everything can change

## Step 5｜What should improve most

1. Feel more premium
2. Feel clearer
3. Feel more modern
4. Convert better
5. Work better on mobile

## Step 6｜New layout direction

1. More minimal
2. More visually striking
3. More steady and professional
4. More product-like

## Step 7｜Navigation style

1. Very simple top navigation
2. Transparent floating navigation
3. Solid fixed navigation
4. Side navigation

## Step 8｜Button style

Use `references/shared-style-library.md`

## Step 9｜Color treatment

1. Mostly keep the existing palette
2. Shift darker for a more premium feel
3. Shift lighter for a cleaner feel
4. Redefine the color system

## Step 10｜Typography update level

1. Mostly keep the current typography
2. Small upgrade
3. Noticeable upgrade
4. Full typography reset

## Step 11｜Card usage

1. Reduce cards where possible
2. Keep some cards
3. Keep the existing card structure
4. Make the layout more card-based

## Step 12｜Imagery update level

1. Mostly keep existing visuals
2. Replace with more premium visuals
3. Add more visual assets
4. Shift toward more product-led imagery

## Step 13｜Motion level

Use `references/shared-style-library.md`

## Step 14｜Device priority

1. Desktop first
2. Mobile first
3. Strong on both
