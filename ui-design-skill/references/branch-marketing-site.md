# Branch B: Conversion Page or Product Website

After the universal questions, ask these steps in order.

## Step 4｜First screen style

1. Full-screen image background
2. Large headline with product visual
3. Minimal oversized headline
4. Split layout with product story on one side and product visual on the other
5. Motion-led or video-led hero

## Step 5｜Navigation style

1. Very simple top navigation
2. Transparent floating navigation
3. Solid fixed navigation
4. Minimize navigation in the first screen

## Step 6｜Hero hierarchy

1. Brand name is largest
2. Product name is largest
3. Value proposition is largest
4. Product visual is largest

## Step 7｜Content density

1. Visual-heavy with very little copy
2. Balanced visuals and copy
3. More complete information while still clean
4. More explanatory and content-rich

## Step 8｜Feature section layout

1. Large visual blocks
2. Card grid
3. Vertical section-by-section explanation
4. Alternating left-right layout
5. Timeline or process-style story

3. Use cards in proof sections only

## Step 16｜Proof or testimonial style

1. Simple metrics or outcomes
2. Customer logo strip
3. Testimonial cards
4. Case study section

## Step 17｜Final CTA style

1. One strong CTA line plus button
2. Lead capture form
3. Contact section
4. Book-a-demo button

## Step 18｜Device priority

1. Desktop first
2. Mobile first
3. Strong on both
