UI Design Skill

让 UI 设计不再困难。这是一个为网页设计、产品官网、落地页和后台界面准备的 UI design skill。它不让用户从空白输入框开始，而是通过一套结构化选项，先快速确定页面类型、视觉方向、信息密度和交互气质。

Quick Install

把 skill 包下载到本地后，可以继续在 Codex、其他 AI 编程 CLI，以及 OpenClaw 这类环境中使用。这里放的是适合首屏展示的示例安装命令，实际目录可以按你的工具环境调整。

cp dist/ui-design-skill.skill "$CODEX_HOME/skills/"

个人网站、产品官网、转化页、应用界面、后台，以及现有页面改版。

01

先问通用问题

先确定页面类型、核心目标和受众，避免一开始就在模糊方向里来回试错。

02

再做具体风格选择

把结构、配色、字体、图片、按钮、信息密度和设备优先级逐项收紧。

03

最后输出完整 prompt

把确认后的方向整理成一份完整 prompt，后续可以继续进入 Variant 或代码工具。

Output Shape
最后交付的是一份可以继续使用的 prompt
Final Prompt

把页面类型、布局方向、视觉系统、信息密度、动效强度和设备优先级整理成一份完整提示词，方便继续生成效果图、页面草案或后续代码。

Privacy Boundary

默认不把内部链接、敏感数据、未发布信息和真实客户资料直接写进第三方工具提示词。
