# UI Design Skill / UI 设计技能

[English](#english) | [中文](#中文)

## English

UI Design Skill is a small GitHub-ready package that includes:

- a one-page landing site
- the `ui-design-skill` skill
- a packaged `.skill` artifact for installation

### Included files

```text
.
├── index.html
├── .vercelignore
├── dist/
│   └── ui-design-skill.skill
└── ui-design-skill/
    ├── SKILL.md
    ├── README.md
    └── references/
```

### What the skill does

The skill helps users avoid starting from a blank prompt box. It runs a compact questionnaire, narrows the UI direction, and produces one complete reusable prompt that can be used in Variant, Codex, GPT-5.4, OpenClaw, and other downstream workflows.

Core search terms this project should naturally match:

- AI UI design
- UI design workflow
- UI design skill
- website design
- landing page design
- product website design
- dashboard UI design

### Website

`index.html` is a one-screen product landing page for the skill. It is designed to deploy cleanly to Vercel as a static site.

### Skill package

The packaged installer lives at:

`dist/ui-design-skill.skill`

The source skill lives at:

`ui-design-skill/`
